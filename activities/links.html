Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute links works for any URL. A relative link is a link that only works for your webpage. You would use a relative link because it doesn't use URL's so if you ever need to change your domain name it is much easier.

3 rules of relative links

  1. Period (.), means current folder
  2. Two periods (..), means parent folder.
  3. Slash (/), means at the root, beggining
