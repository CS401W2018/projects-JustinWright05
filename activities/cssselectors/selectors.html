Different ways to select an item

ID tag, Class

What is the difference between an ID and a Class?

Every ID should be unique. Same as naming someone. You can only have one ID tag of the same name. Classes are fine to reuse. ID is a noun. Class is adjective. Apply multiple tags on class not on ID.

What is the difference between a SPAN tag and a DIV tag?

Span is for grouping words and sentences. Div tag is for grouping a large group of words like p tags. It creates a large box that puts things aboive it on top and below under.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

Div is editable for any tag. Whereas header, main, and footer stay in their respected areas
