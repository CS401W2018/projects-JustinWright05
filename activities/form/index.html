The form tag has an "action" attribute. What does it do?

It is the place that we are being sent to when we submit a form.

The form tag has a "method" attribute. What does it do?

Part of the http protocol. Helps with the server and the website communicating with each other.

What is the difference between a "POST" and a "GET"?

GET shows all written things on the form inside of the website address. It's when we are reading information. Used with search bars on google. POST does not do that. It is for changing information on the website.
