Explain the difference between inline, embedded, and external CSS.

Inline is in the tags. For example it would be in a parahraph tag. Embedded is in the head of the HTML. External is a .css file seperate from the HTML file.

Why should we use External CSS?

Because any of the other ways will overide external css. So if you were working with someone who is programming the HTML file and they use an inline tag, the person operating the external css can't make changes to what the inline tag made.

What is the Selector used for

The selector is used to single out specific tags to style.
