Internet

A global network of connected computers that lets people share information and access websites. Allows you to send emails, use social media, and watch videos from anywhere as long as you are connected to the Internet.

Web

Short for World Wide Web, Web is the list of websites and Web pages that are on the internet. It uses browsers like Safari and Google Chrome to show text, images, videos, etc. Web runs on a system of links called hyperlinks that connect Webpages. This allows you to easily navigate between them.

How are the Internet and Web different?

The Internet is a physical network of devices around the world that are all connected to each other. The Web is a type of service that can accessed through the internet using a web browser. The difference is that the Internet is the global network and the Web is one of the many things that can be accessed through the internet.
